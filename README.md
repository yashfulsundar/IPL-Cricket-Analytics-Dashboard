# IPL Cricket Analytics Dashboard

## Project Overview

An interactive IPL Cricket Analytics Dashboard built using Excel, MySQL, and Power BI.

The project analyzes IPL ball-by-ball data to generate insights on batting performance, bowling performance, team statistics, venue trends, and season-wise analysis.

---

## Tools Used

- Excel
- MySQL
- Power BI

---

## Project Workflow

Dataset Collection
→ Data Cleaning in Excel
→ SQL Database Creation
→ SQL Analysis
→ Power BI Dashboard Development

---

## Dashboard Pages

### 1. Overview Dashboard
- Total Matches
- Total Runs
- Total Wickets
- Total Sixes
- Team Performance
- Season-wise Trends

### 2. Batting Analysis
- Top Run Scorers
- Most Sixes
- Batting Performance Analysis
- Orange Cap Insights

### 3. Bowling Analysis
- Top Wicket Takers
- Wicket Type Distribution
- Economy Rate Analysis
- Purple Cap Insights

---

## SQL Analysis Performed

- Top 10 Run Scorers
- Top 10 Wicket Takers
- Team Performance Analysis
- Venue Analysis
- Sixes Analysis
